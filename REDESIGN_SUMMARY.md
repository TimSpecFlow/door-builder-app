- `.door-showcase` - Container for door style showcase

- `.showcase-image` - Image wrapper with hover effects
- `.section-portfolio` - Portfolio gallery section
- `.portfolio-grid` - Masonry grid layout
- `.portfolio-item` - Individual portfolio item
- `.portfolio-overlay` - Hover overlay with text

### Images Used
All images are from Unsplash (free, high-quality stock photos):
- Door styles: Various professional interior door photography

### Responsive Breakpoints
- Desktop: 3-column portfolio grid
- Tablet (768px): 2-column grid, adjusted showcase
- Mobile (640px): Single column layouts
